# Milestone 1 – Environment Infrastructure

## Purpose

This engineering binder documents the deployment of the virtual infrastructure supporting the Northwind Technologies Enterprise IT Modernization Project.

The procedures contained within this milestone describe how the virtualization platform, operating systems, validation testing, and troubleshooting activities were completed to establish the enterprise environment used throughout Sprint 1.

## Contents

- Oracle VirtualBox Configuration
- Windows Server 2025 Installation
- Windows 11 Enterprise Installation
- Ubuntu Server Installation
- Environment Validation
- Troubleshooting

## Prerequisites

- Oracle VirtualBox installed
- Installation media available
- Host workstation meets minimum hardware requirements

## Outcome

Upon completion of this milestone, the enterprise virtualization platform is operational and ready for Active Directory deployment.
