# Executive Summary

The Enterprise IT Support Lab is a comprehensive enterprise infrastructure project designed to simulate the planning, deployment, administration, and operational support of a modern corporate IT environment.

Rather than demonstrating isolated technical exercises, this project documents the complete lifecycle of an enterprise network beginning with infrastructure planning and continuing through deployment, operational support, automation, security monitoring, cloud administration, and documentation.

The environment is intentionally built using industry-standard technologies that closely mirror production environments found within medium and large organizations.

Primary technologies include:

* Windows Server 2025
* Active Directory Domain Services
* DNS
* DHCP
* Enterprise File Services
* NTFS Permissions
* Group Policy
* Windows 11 Enterprise
* Ubuntu Server
* osTicket Help Desk
* PowerShell Automation
* Microsoft 365 (Planned)
* Wazuh SIEM (Planned)

---

# Business Purpose

Organizations rely on standardized infrastructure, documented procedures, and repeatable administrative processes to maintain secure, scalable, and reliable IT operations.

The purpose of this project is to demonstrate practical experience designing, implementing, documenting, securing, and supporting enterprise infrastructure using technologies commonly encountered by Help Desk Technicians, Systems Administrators, and Infrastructure Engineers.

This repository serves as both a technical portfolio and an operational knowledge base.

---

# Project Objectives

The project will accomplish the following objectives:

* Deploy a fully functional Active Directory environment
* Implement enterprise DNS and DHCP services
* Design a realistic Organizational Unit structure
* Create departmental users and security groups
* Configure enterprise file shares and NTFS permissions
* Deploy an internal Help Desk platform using osTicket
* Simulate Tier 1 and Tier 2 support operations
* Document enterprise administrative procedures
* Implement PowerShell automation
* Deploy Microsoft 365 hybrid administration
* Implement enterprise security monitoring
* Produce professional documentation suitable for technical portfolio review
