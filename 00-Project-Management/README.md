# Project Management

Welcome to the Project Management documentation for Northwind Technologies.

This section contains the planning, governance, standards, and project oversight documentation used throughout the Enterprise IT Support Lab.

Rather than building isolated technical labs, this project follows a structured implementation methodology that mirrors how enterprise IT infrastructure projects are planned and executed in production environments.

The objective is to demonstrate not only technical implementation skills, but also project planning, documentation, change management, and operational decision-making.

---

## Project Documents

### Project Charter

Defines the purpose, objectives, scope, stakeholders, and success criteria for the project.

### Roadmap

Provides a high-level timeline showing the implementation sequence of all seven project sprints.

### Milestones

Documents major accomplishments completed throughout the project.

### Risks

Identifies technical, operational, and project risks together with mitigation strategies.

---

## Project Methodology

The project is implemented using a phased sprint approach.

* Sprint 1 – Enterprise Infrastructure
* Sprint 2 – Enterprise Operations
* Sprint 3 – Help Desk Operations
* Sprint 4 – Security Operations
* Sprint 5 – Cloud Administration
* Sprint 6 – Automation
* Sprint 7 – Enterprise Capstone

Each sprint includes planning, implementation, validation, troubleshooting, and documentation before progressing to the next phase.

## Deliverables

The final repository will include:

* Enterprise network design
* Active Directory implementation
* Enterprise file services
* DHCP and DNS configuration
* Group Policy administration
* Help Desk ticketing system
* Microsoft 365 administration
* PowerShell automation
* Standard Operating Procedures
* Incident response documentation
* Enterprise architecture diagrams

## Success Criteria

The project will be considered complete when all planned infrastructure has been deployed, documented, validated, and organized into a professional GitHub portfolio demonstrating production-inspired enterprise administration skills.
